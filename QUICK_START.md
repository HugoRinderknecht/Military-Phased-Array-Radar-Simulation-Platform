│  🚀 启动方法                                                       │
└───────────────────────────────────────────────────────────────────┘

    Windows 用户:
        双击 start-backend-only.bat  (仅后端，推荐)
        或双击 start.bat              (后端+前端)

    Linux/Mac 用户:
        chmod +x start.sh
        ./start.sh

    注意: start.bat 必须使用纯英文编码，请勿用记事本编辑！

    Q: 启动脚本闪退？
    A: 已修复！现在 start.bat 会自动创建虚拟环境和安装依赖

    Q: 提示缺少 Python？
    A: 请安装 Python 3.8+ https://www.python.org/downloads/

    Q: 端口 8000 被占用？
    A: 关闭其他占用该端口的程序，或修改 .env 中的端口号

    Q: 依赖安装失败？
    A: 检查网络连接，或使用国内镜像源:
       pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
